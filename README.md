
# Online High School Student Management System (OHSSMS)

This is a comprehensive web-based platform designed to streamline administrative tasks, manage student records, and facilitate communication between teachers and students in a high school environment.

## 🛠 Tech Stack
* **Backend:** PHP (Server-side logic)
* **Database:** MySQL / MariaDB
* **Local Server:** XAMPP (Apache)
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap

## 🚀 Key Features
* **Student Information Management:** Digital profiles for all enrolled students.
* **Academic Tracking:** Manage grades, subjects, and exam results.
* **Attendance System:** Daily tracking of student presence/absence.
* **User Roles:** Separate portals for Administrators, Teachers, and Students.
* **Secure Login:** Password-protected access to sensitive student data.

## 💻 How to Install Locally
1. Clone this repository to your XAMPP `htdocs` folder.
2. Open the **XAMPP Control Panel** and start **Apache** and **MySQL**.
3. Create a database named `ohssms_db` in **phpMyAdmin**.
4. Import the provided SQL file (e.g., `database.sql`) into that database.
5. Open your browser and visit: `http://localhost/OHSSMS/`
